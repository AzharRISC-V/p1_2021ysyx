## 01 - 锁存器、触发器、寄存器、移位器

* 锁存器、触发器、寄存器的区别
  * 锁存器（Latch）是对电平敏感的存储单元电路
  * 触发器（Flip-Flop，简称FF）是对边沿敏感的存储电路
  * N个触发器（锁存器）组成寄存器
* 同步清零、同步复位；异步清零；异步复位
  * 同步是指，在clk变化时，才采样和使用set、reset控制信号。
  * 异步是指，不管clk在任何位置，set、reset都能立即执行。
